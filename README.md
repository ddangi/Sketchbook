# Sketchbook 🎨

손으로 그린 그림을 올리고 감상할 수 있는 개인 갤러리 웹사이트입니다.

## 폴더 구조

```
sketchbook/
├── index.html          # HTML 뼈대
├── css/
│   └── style.css       # 전체 스타일
├── js/
│   ├── main.js         # 앱 진입점 (각 모듈 초기화)
│   ├── config.js       # 사이트 설정 (비밀번호 등)
│   ├── auth.js         # 관리자 로그인/로그아웃
│   ├── gallery.js      # 갤러리 그리드 렌더링
│   ├── upload.js       # 사진 업로드 패널
│   ├── lightbox.js     # 라이트박스 + 댓글
│   ├── storage.js      # 데이터 저장/불러오기
│   └── utils.js        # 공통 유틸리티
└── assets/             # 폰트, 아이콘 등 정적 파일
```

## 로컬에서 실행하기

JS 모듈(`type="module"`)을 사용하므로 **로컬 서버**가 필요합니다.

##사이트 기능
- 모바일과 pc 웹에서 접근 가능한 반응형 웹사이트를 만든다.
- 손으로 그린 그림을 사진으로 찍어서 올려 개인적으로 기록하는 용도의 사이트다.
- 아이폰 사진첩에 있는 사진을 업로드할 수 있다.
- 한 번에 여러장의 사진을 선택해서 올릴 수 있다.
- 사진 업로드시 간단한 메모를 작성할 수 있다.
- 업로드된 게시물은 사진을 보기 좋은 ui로 표시한다.
-- 인스타그램이나 사진첩 ui처럼 사진 목록이 3x3 격자 형태로 표시되는 형태
-- 한 번에 선택해서 올린 사진이 여러장인 경우 첫번째 사진이 목록에 표시된다.
- 게시물 목록 중 하나를 선택하면 해당 게시물 화면으로 이동되며 더 큰 화면으로 사진을 볼 수 있고 작성해둔 메모가 표시된다.
-- 인스타 그램 ui와 동일한 형태
- 게시물 화면에서는 한번에 업로드한 사진이 여러장이었다면 대표 사진 하단에 업로드한 사진 목록이 작게 표시되고 다른 사진을 선택하면 선택한 사진을 크게 볼 수 있도록 한다.
- (목록 페이지 말고) 게시물 화면에서 다른 사람들은 자유롭게 댓글을 작성할 수 있다.
- 이 웹사이트에서 사진은 나만 올릴 수 있다. (관리자 비번으로 로그인)

##고쳐야 하는 버그
- 아이폰 사파리 웹에서 '올리기' 버튼을 누르면 여전히 아무 동작하지 않는다.(pc에서는 잘 작동함)
- 관리자 로그인이 자꾸 풀린다. 
- 관리자 로그인하지 않은 상태라면 '사진 올리기' 버튼이 노출되면 안된다.
- 왼쪽 상단의 'SketchBook' 로고를 클릭하면 항상 '홈화면(사진 목록 화면)'으로 이동해야 한다.
- 올린 사진이 로컬에 저장되지 않도록 백엔드로 교체해야 함. 

##향후 개선 방안
- 회원가입을 지원
- 관리자 패스워드고 코드에 포함되어 있는 보안 문제 해결하기(현재는 그냥 연습이라 감수하기로 함)

## 데이터 저장 방식

현재는 브라우저 **localStorage**에 저장됩니다.
여러 기기에서 동기화하려면 `js/storage.js`를 Firebase 등 백엔드로 교체하면 됩니다.
